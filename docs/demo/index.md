# Live Demo

See neoprint in action. Click the button below to generate your browser's fingerprint in real-time.

Everything runs **client-side** — no data is sent to any server.

<LiveDemo />

## What you're seeing

- **Overview** — Your 4 fingerprint IDs, confidence score, entropy, and environment details
- **Components** — Raw results from each of the 19 collectors with timing and entropy data
- **Security** — Bot detection, anti-detect browser detection, and privacy tool analysis
- **Benchmark** — How long each collector takes on your device
- **Raw** — The full fingerprint JSON object

::: tip Try these experiments
1. Open this page in **another browser** and compare the Cross-Browser ID
2. Open in **incognito mode** and compare the Stable ID
3. Use a **VPN or proxy** and see what changes
4. Try with **Brave** to see noise detection in action
:::
